# git Log Command

최신 버전 확인 : 
커밋 해시 끝 부분에 (HEAD -> master) 이 적혀있으면, 최신버전

Autor : 
작성자 

Date : 
버전 생성 날짜

Commit Message :
커밋할 때 입력한 메세지
없는 경우도 있음 (메세지 입력 안해서 없는거)


# Option ----------

--stat :
커밋과 관련된 파일까지 함께 보여줌
로그 메세지가 많을 경우 한 화면씩 나눠서 보여줌
enter 키로 다음 로그화면을 볼 수 있고, Q키를 눌러 종료


# HEAD ------------
현재 작업중인 커밋을 나타냄
HEAD -> master 라는것은, 현재 작업중인 커밋이 최신커밋(master)라는 것
